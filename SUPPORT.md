## Reporting Issues

### Before Reporting

1. Search existing issues to avoid duplicates
2. Check the FAQ for common solutions
3. Gather relevant information

### Bug Report Template

When reporting bugs, include:
- NexusAI ERP version
- Browser and OS
- Steps to reproduce
- Expected vs actual behavior
- Screenshots if applicable
- Error messages or logs

### Security Issues

For security vulnerabilities, please do NOT create public issues.
See our [Security Policy](./SECURITY.md) for responsible disclosure.

## Self-Help Resources

### Troubleshooting Steps

1. **Check logs** - Review browser console and server logs
2. **Verify configuration** - Ensure environment variables are set
3. **Clear cache** - Try clearing browser cache
4. **Restart services** - Restart the development server
5. **Check database** - Verify database connection

### Common Issues

| Issue | Solution |
|-------|----------|
| Cannot connect to database | Check DATABASE_URL in .env |
| Page not loading | Clear browser cache, check console |
| Authentication failing | Verify session configuration |
| API errors | Check server logs for details |
