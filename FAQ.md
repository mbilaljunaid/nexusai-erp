# Frequently Asked Questions

## General Questions

### What is NexusAI ERP?

NexusAI ERP is an open-source, AI-powered enterprise resource planning platform designed to streamline business operations across 40+ industries. It includes 12 core modules covering CRM, Finance, HR, Manufacturing, Supply Chain, and more.

### Is NexusAI ERP free to use?

Yes! NexusAI ERP is released under the AGPL-3.0 license, making it free to use, modify, and distribute. The only requirement is that modifications must also be open-sourced under the same license.

### What industries does it support?

NexusAI ERP supports 40+ industries including Manufacturing, Retail, Healthcare, Finance, Construction, Technology, Education, and many more. Each industry has pre-configured templates and workflows.

## Technical Questions

### What are the system requirements?

- Node.js 18+
- PostgreSQL 12+
- 2GB RAM minimum
- Modern web browser

### How do I set up the development environment?

See our [Installation Guide](./INSTALL.md) for step-by-step instructions.

### Can I deploy on my own servers?

Yes! NexusAI ERP can be deployed on:
- Replit (one-click deployment)
- Docker containers
- Kubernetes clusters
- Any VPS or cloud provider

### Does it work offline?

Currently, NexusAI ERP requires an internet connection. Offline mode is planned for a future release.

## Licensing Questions

### What is the AGPL-3.0 license?

The GNU Affero General Public License v3.0 allows you to:
- Use the software freely
- Modify the source code
- Distribute copies
- Run as a network service

With the requirement that:
- Modifications must be open-sourced
- Network users must have access to source code
- Same license must be used for derivatives

### Can I use it for commercial purposes?

Yes, you can use NexusAI ERP commercially. However, any modifications must be released under AGPL-3.0.
