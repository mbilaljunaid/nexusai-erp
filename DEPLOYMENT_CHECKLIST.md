### Deployment Instructions

1. **Prerequisites:**
   - Node.js 20+
   - PostgreSQL database
   - Redis instance

2. **Environment Variables:**
   - DATABASE_URL
   - SESSION_SECRET
   - NODE_ENV=production

3. **Build:**
   ```bash
   npm run build
   ```

4. **Start:**
   ```bash
   npm run start
   ```

5. **Access:**
   - Frontend: https://your-domain.com
   - API: https://your-domain.com/api
